# Sales Performance Dashboard

This is a Power BI dashboard I built using the Superstore dataset from Kaggle to practice data analysis and dashboard design. The goal of this project was to explore sales performance, identify profitable and loss-making areas, and present the findings in a simple, interactive dashboard.

## Dataset

- Superstore Dataset (Kaggle)

## Tools Used

- Power BI
- SQL
- DAX
- Power Query

## Dashboard Highlights

### Overview
- Total Sales: **2.30M**
- Total Profit: **286.4K**
- Profit Margin: **12.47%**
- Monthly sales trend
- Average shipping days by shipping mode
- Profit impact across different discount ranges

### Sales Analysis
- Sales contribution by category
- Top-performing states
- Products with high sales but negative profit

### Customer & Regional Analysis
- Top 10 customers by sales
- Sales and profit by region
- Top 5 most profitable sub-categories

## What I Found

- Sales increased steadily over the years.
- California generated the highest sales.
- Technology was the largest sales category.
- Discounts above 20% often resulted in losses.
- Some products had strong sales but consistently lost money.
- The West region generated the highest profit.
- Copiers were the most profitable sub-category.
